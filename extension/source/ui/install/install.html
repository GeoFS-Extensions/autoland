Context

For anyone that doesn't know anything about this extension or GeoFS, let's rewind the clock. Keep in mind this is a very brief summary, and isn't the full story.

Back in 2010, Xavier Tassin released GEFS (Google Earth Flight Simulator). In 2016, in light of the termination of the plugin GEFS used, GEFS migrated to CesiumJS, an open source virtual globe platform. It was then when GEFS becaume GeoFS. This is when its community started taking off. Developers started creating plugins for GeoFS. Two major players in the field of GeoFS plugins were Harry Xue and Karl Cheng. They both created plugins that were later abandoned, AP++ and FMC. In summer of 2020, a project emerged, which was at the time called GeoFS Autoland. This project eventually evolved into current-day GeoFS Plugins. This extension builds on top of the now unmaintained plugins, through more detailed control and the addition of new features and scripts.
